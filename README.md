# 🔵 HoloMatrix — Particle Hologram Video Converter

Turn any video into a stunning real-time holographic particle effect.

## Features
- 6 hologram color modes (Cyan, Plasma, Matrix, Aurora, Ghost, Gold)
- Real-time particle rendering from video frames
- Adjustable density, size, drift, glow, and threshold
- Video transport controls (play, pause, seek, mute)

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Click Deploy — done!

## Tech Stack
- React 18
- Vite
- HTML5 Canvas API
